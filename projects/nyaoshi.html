Nyaoshi

A cute desktop app that watches for new photos and uploads them to Myoshi.co

Role: Sole Developer

About Nyaoshi

Nyaoshi (a playful cat-themed twist on Myoshi) is a desktop utility that makes uploading photos to your Myoshi albums effortless. Just drop images into a watched folder and they'll be queued up for upload automatically.

Built with a focus on being cute, lightweight, and unobtrusive - Nyaoshi lives in your system tray and quietly watches for new photos, uploading them in the background or queuing them for your review.

Download Myoshi.co

Features

Directory Watching

Monitor a folder and all its subfolders for new images. When a new photo is detected, it's automatically added to the queue.

Auto-Upload Mode

Set a default album and let Nyaoshi automatically upload new photos without any interaction required.

Queue Mode

Review photos before uploading. Approve or skip each photo and select which album to upload to.

Events Timeline

View upcoming Myoshi events in a beautiful timeline format with live/soon status badges.

Auto-Scaling

Large images over 25MB are automatically scaled down to meet upload requirements.

System Tray

Runs quietly in the background. Left-click to show, right-click for quick actions like quit.

Technologies

Nyaoshi is built with modern technologies for a fast, native experience.

Tauri v2

Rust-based framework for building lightweight, secure desktop applications with web technologies.

Rust

Backend logic including file watching, image processing, and API communication - all in safe, performant Rust.

SvelteKit

Modern frontend framework with Svelte 5 runes for reactive, component-based UI development.

TypeScript

Type-safe frontend code with full IDE support and compile-time error checking.

GraphQL

Integration with Myoshi.co's GraphQL API for fetching albums, events, and uploading photos.

notify (Rust)

Cross-platform file system notification library for efficient directory watching.
